    BenditDevice()

    An object that represents a Bendit board/circuit-bent device pair. New devices are added to the Bendit.devices array on creation.

    The socket.io socket inhereted from the global Bendit-class instance.
    deviceNickname string

    A name to associate with this circuit-bent device/Bendit board pair (e.g. "Walkman" or "Casio keyboad")
    deviceColor string

    Color assigned to the associated Bendit board's LED by the server.

    Hardware version of the associated Bendit board.

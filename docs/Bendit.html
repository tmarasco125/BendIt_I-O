    Bendit(url)

    To initialize a connection to the server, an instance of the Bendit class needs to be initialized first by declaring a new Bendit.Browser(). This class handles information about the client's connection to the server as well as information about all client and Bendit board activity on the server.

    new Bendit(url)
    benditBrowser.js, line 83

    Parameters:
    Name Type Description
    url string

    [optional] The address of the Bendit_I/O server. Defaults to the host address of the page.
    Properties:
    Name Type Description
    users Array.<string>

    An array of connected client websocket IDs
    devices Array.<Object>

    An array of created BenditDevices that can assigned to connected Bendit boards
    socket Object

    The socket.io socket for this Bendit-class instance. Read only.
    availableBoards Array.<Object>

    An array of assigned board data for all connected Bendit Boards
    messageFromBoard String

    Current message received from a Bendit board.
    Example
    //load Bendit modules (require can be used)
let Bendit = require('bendit');

//start the connection to the server
let bendit = new Bendit.Browser();

    addDevice(switches, pots, motors, boardNumber) → {Object}
    benditBrowser.js, line 213

    Creates an instance of the BenditDevice class and adds that object to the Bendit.devices array. Can be called with individual arguments or with an object.

    Parameters:
    Name Type Description
    switches number

    the total number of switches to assign to the device.
    pots number

    the total number of switches to assign to the device.
    motors number

    the total number of motors to assign to the device.
    boardNumber number

    the Bendit board number to assign to the device.
    Returns:
    Object -
    • An instance of the BenditDevice class.
    Example
    //Create Bendit instance
let bendit = new Bendit.Browser();

let speakNspell = bendit.addDevice(6,5,1,2);// possible switches, pots, and motors to control on board 2

or

let speakNspell = bendit.addDevice({
      "switches": 6,
      "pots": 5,
      "motors": 1,
      "boardNumber": 2     
      });

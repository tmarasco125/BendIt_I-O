    Motor(motNum, socket, deviceNum)

    An object that represents a motor output channel on a Device's associated Bendit board/circuit-bent device pair. An array of Motor objects are created when a new BenditDevice object is created.

    new Motor(motNum, socket, deviceNum)

    The motor channel of this device 's Bendit board.
    socket Object

    The socket.io socket inhereted from the global Bendit - class instance. Do not change this.
    deviceNum number

    The number of the device associated with this motor. Passed on to be associated with the Bendit board this motor's device is assigned to.

    The motor channel of this device's Bendit board.

    The number of the Bendit board that this device and this motor are associated with.

    Methods

    flipDirection()
    benditBrowser.js, line 787

    For DC motors. Reverses this channel's motor on this device's Bendit board. Speed is kept at most recently set value. Motors in the BenditDevice.motors array are zero indexed.

    Example
    console.log(walkman.motors[1].direction); //--> prints 1, aka forward

//Flip the direction of Motor channel 2 on this Bendit board

walkman.motors[1].flipDirection(); //Motor 2 is now running backwards, aka -1

//Flip the direction of Motor 2 channel back

walkman.motors[1].flipDirection(); //Motor 2 is now running forwards, aka 1

    run(speed, direction)
    benditBrowser.js, line 729

    For DC motors. Starts this channel's motor on this device's Bendit board. Motors in the BenditDevice.motors array are zero indexed.

    Parameters:
    Name Type Description
    speed number

    The speed of the motor. Range is 0 (stopped) to 255 (ful speed).
    direction number

    The direction of the motor. Options are 1 (forward) and -1 (backwards).
    Example
    //Set speed and direction of Motor channel 1 on this Bendit board to forward at 127 (half-speed)
 tapeDeck.motors[0].run(127, 1);

//Set speed and direction of Motor channel 1 on this Bendit board to backwards at 64 (quarter-speed)
tapeDeck.motors[0].run(64, -1);

    Pot(potNum, socket, deviceNum)

    An object that represents a potentiometer output channel on a Device's associated Bendit board/circuit-bent device pair. An array of Pot objects are created when a new BenditDevice object is created.

    new Pot(potNum, socket, deviceNum)
    benditBrowser.js, line 581

    Parameters:
    Name Type Description
    potNum number

    The potentiometer channel of this device's Bendit board.
    socket Object

    The socket.io socket inhereted from the global Bendit - class instance. Do not change this.
    deviceNum number

    The number of the device associated with this pot. Passed on to be associated with the Bendit board this pot's device is assigned to.
    Properties:
    Name Type Description
    number number

    The potentiometer channel of this device 's Bendit board.
    position number

    The position of the pot. Range is 0 to 255 (0 ohms to 100k ohms).
    socket Object

    The socket.io socket inhereted from the global Bendit - class instance. Do not change this.
    boardNumber number

    The number of the Bendit board that this device and this pot are associated with.

    setPot(v)
    benditBrowser.js, line 616

    Sets the position of the pot on this device's Bendit board. Pots in the BenditDevice.pots array are zero indexed.

    Parameters:
    Name Type Description
    v number

    The position of the switch, corresponding to the generated resistance level. Range is 0 - 255, corresponding to a resistance range of 0 to 100k ohms.
    Example
    //Set position of Pot channel 3 on this Bendit board to 127 (midway)
 vhs.pots[2].setPot(127); //circa 50k ohms generated on Pot channel 3
